How to publish the update

  1. In the Core Data Sheet: File → Download → Comma-separated values (.csv).
  2. Drop that CSV above and click Build, then Download fish_index.json.
  3. Go to the fishtagged-api repository on GitHub, open fish_index.json, and replace it with the new one. Commit.
  4. The live search updates by itself in about a minute. Search a brand-new tag number to confirm.
